## Communication Plan

The group will use WhatsApp, Zoom, GitHub, and tutorial time for coordination. WhatsApp will be used for fast updates, screenshots, and small issues. Zoom will be used when the team needs to review OpenWRT, firewall results, packet captures, diagrams, or final files together.

Key communication points:

- Share progress twice each week.
- Upload screenshots with correct names.
- Check GitHub updates regularly.
- Discuss errors before major changes.
- Review files before final submission.
